Tailwind configuration
----------------------

put tailwindcss.exe at web-adapter root


at PowerShell run
./tailwindcss init


at tailwind.config.js edit
content: ["./src/main/resources/templates/**/*.{html,js}"]


at PowerShell
./tailwindcss -i ./src/main/resources/static/css/input.css -o ./src/main/resources/static/css/output.css --watch


at index.html
<link rel="stylesheet" th:href="@{./css/output.css}" >
